# Hífen (`-`)

O hífen representa o último diretório acessado. Veja um exemplo simples. Execute os seguintes comandos:

1. `cd /usr/share/doc`
2. `pwd`
3. `ls`
4. `cd /etc/init.d`
5. `pwd`
6. `ls`
7. `cd -`
8. `pwd`
9. `cd -`
10. `pwd`

Os comandos de número 1, 4, 7 e 9 servem para mudar de diretório. Observe a saída deles.
